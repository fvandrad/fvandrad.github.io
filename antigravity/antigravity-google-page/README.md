# Google Antigravity Page

This project is a concept landing page for **Google Antigravity**, the new Agent-First IDE powered by Gemini 3.

## Features
- **Modern Dark Mode Design**: Inspired by developer tools and premium software.
- **Glassmorphism**: sleek, translucent UI elements.
- **Animations**: Floating elements and smooth scroll reveals.
- **Responsive**: Works on desktop and mobile.

## Structure
- `index.html`: Main structure.
- `style.css`: Styling and animations.
- `script.js`: Interactive effects.
- `assets/`: Contains images.

## How to Run
Open `index.html` in your browser.
